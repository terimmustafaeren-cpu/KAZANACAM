KAZANACAM v3 - demo package
Files:
- index.html
- styles.css
- script.js
- assets/logo.svg
- assets/matematik_6_1.pdf (the PDF you uploaded; opens in-browser)

How to use:
1. Unzip and open index.html locally to preview.
2. To display the uploaded PDF for 6th grade Math, click the 6th grade -> Matematik -> Konu and then 'PDF Göster'.
3. To add YouTube videos: paste each YouTube URL on a new line in the "Videoları ekle" textarea and click "Kaydet".
4. To publish: upload these files to a GitHub repository (root, not inside a folder) and enable GitHub Pages.
